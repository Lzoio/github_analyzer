# GitHub Analyzer 🔍

<img src="https://narwal.ai/wp-content/uploads/2023/11/New-Project-6.jpg" alt="Data Analyzer" width="500" height="300">

**GitHub Analyzer** é uma ferramenta poderosa projetada para fornecer insights profundos sobre repositórios do GitHub. Seja você um desenvolvedor buscando auditar seu próprio código ou um recrutador tentando entender a complexidade técnica de um candidato, este projeto automatiza a extração e análise de dados de repositórios.

## 🚀 Funcionalidades

- **Descoberta de Repositórios:** Busca automaticamente repositórios públicos de um usuário específico do GitHub.
- **Avaliação Técnica:** Analisa a complexidade do código, algoritmos e a saúde do repositório.

## 🚀 Funcionalidades Futuras (Em desenvolvimento)
- **Resumos com IA:** Utiliza LLMs (como GPT) para fornecer resumos inteligentes sobre os objetivos do projeto.
- **Visualização de Dados:** Gera relatórios sobre a distribuição de linguagens e atividade de contribuidores.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python
- **IA/LLM:** LangChain & OpenAI API
- **Frameworks:** Streamlit (para a interface de usuário)
- **Manipulação de Dados:** Pandas & NumPy
- **Armazenamento:** FAISS (para armazenamento de vetores/embeddings)

## ⚙️ Instalação e Configuração

Siga estes passos para configurar uma cópia local do projeto.

- Um Token de Acesso Pessoal do GitHub (Personal Access Token)


**GitHub Analyzer** is a powerful tool designed to provide deep insights into GitHub repositories. Whether you are a developer looking to audit your own code or a recruiter seeking to understand a candidate's technical complexity, this project automates the extraction and analysis of repository data.

## 🚀 Features

- **Repository Discovery:** Automatically fetches public repositories from a specific GitHub user.
